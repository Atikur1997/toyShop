**Toy Store Platform** is a modern web application built with **React** and **Vite**, designed to provide an interactive user experience for browsing and purchasing toys online.

## ğŸ”‘ Key Features

### 1. Product Browsing & Discovery
- Browse a wide variety of toys with images, descriptions, and prices.  
- Filter and search functionality to quickly find desired toys.  
- Real-time UI updates when adding to favorites or cart.  

### 2. User Authentication
- Firebase Authentication for secure signup and login.  
- Support for email/password login and Google authentication.  

### 3. Favorites & Cart Management
- Users can like/favorite toys for quick access later.  
- Manage shopping cart for selecting toys before checkout.  

### 4. Notifications & Feedback
- Instant toast notifications for actions like adding items to favorites or cart.  
- Feedback system to inform users about successful operations or errors.  

### 5. Smooth UI & Animations
- Fully interactive interface with AOS scroll animations.  
- Responsive design for desktop, tablet, and mobile devices.  

### 6. Tech & Data Security
- **Frontend:** React + Vite + TailwindCSS + DaisyUI + Lucide React + React Icons.  
- Real-time updates powered by Firebase.  
- Data security ensured with Firebase Authentication.  

### 7. Deployment & Scalability
- Hosted on Netlify for frontend.  
- Scalable architecture to support growing users and product listings.

## ğŸš€ Getting Started

### Prerequisites
- Node.js v18+  
- npm or yarn  

### Frontend Setup
```bash
# Clone the client repo
git clone https://github.com/Atikur1997/toy-store-platform.git

# Navigate to client directory
cd toy-store-platform

# Install dependencies
npm install

# Create a .env file for Vite (optional)
# This file can store your Firebase configuration
echo "VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_firebase_auth_domain
VITE_FIREBASE_PROJECT_ID=your_firebase_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_firebase_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_firebase_messaging_sender_id
VITE_FIREBASE_APP_ID=your_firebase_app_id" > .env

# Start the development server
npm run dev
```
